### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

Create `.env` file:
```

```

```bash

```

**Main Interface** (for using AI):
http://localhost:8000

**Advanced Dashboard** (shows internal workings):
http://localhost:8000/dashboard
